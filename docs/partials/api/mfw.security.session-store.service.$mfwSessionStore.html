$mfwSessionStore
service in module mfw.security.session-store

Description

Session Store that notifies session changes: login and logout.

It handles the following flux actions:

  • $mfwSessionEventConstant.LOGIN: automatically fires mfw:Session.login event.
  • $mfwSessionEventConstant.LOGOUT: automatically fires mfw:Session.logout event.
  • $mfwSessionEventConstant.UPDATE_CREDENTIALS: automatically fires mfw:Session.credentials-update event.
