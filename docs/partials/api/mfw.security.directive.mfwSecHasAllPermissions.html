mfwSecHasAllPermissions
directive in module mfw.security

Description

Display HTML content when current logged user contains all the specified permissions.

Usage

as attribute
<ANY mfw-sec-has-all-permissions="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mfwSecHasAllPermissions|fromString

Comma-separated list of permission names
