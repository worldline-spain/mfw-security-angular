$mfwSessionEventConstant
object in module mfw.security.session-store

Description

Available flux events fired by $mfwSessionAction action creator, handled by $mfwSessionStore.

Properties

  • LOGIN

    Constant value to be used with $scope.$listenTo() to be notified of login events.

    Default value mfw:Session.login.

    Returns
    String

    Login event name.

  • LOGOUT

    Constant value to be used with $scope.$listenTo() to be notified of logout events.

    Default value mfw:Session.logout.

    Returns
    String

    Logout event name.

  • SESSION_ALL

    Constant value to be used with $scope.$listenTo() to be notified of all session events.

    Default value mfw:Session.*.

    Returns
    String

    Session events wildcard.

  • UPDATE_CREDENTIALS

    Constant value to be used with $scope.$listenTo() to be notified of update credentials events.

    Default value mfw:Session.credentials-update.

    Returns
    String

    Updated credentials (refresh token) event name.
