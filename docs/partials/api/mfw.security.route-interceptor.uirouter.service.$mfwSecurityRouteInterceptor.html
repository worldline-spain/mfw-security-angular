$mfwSecurityRouteInterceptor
service in module mfw.security.route-interceptor.uirouter

Description

Service that listens to UI Router events ($stateChangeStart and $stateChangeError) to implement authentication and authorization logic.

Methods

  • init()

    Initializer method, to be called when module is loaded.

    It register for UI Router events and $mfwSecurity's events.

Events

  • $mfwSecurityStateTransitionStopped

    Event triggered when a state transition is stopped due to missing user permissions.

    Type:

    broadcast

    Target:

    $rootScope
