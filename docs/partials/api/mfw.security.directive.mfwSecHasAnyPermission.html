mfwSecHasAnyPermission
directive in module mfw.security

Description

Display HTML content when current logged user contains at least one of the specified permissions.

Usage

as attribute
<ANY mfw-sec-has-any-permission="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mfwSecHasAnyPermission|fromString

Comma-separated list of permission names
