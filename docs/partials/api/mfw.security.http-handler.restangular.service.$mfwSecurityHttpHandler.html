$mfwSecurityHttpHandler
service in module mfw.security.http-handler.restangular

Description

Service that listens to HTTP error responses by registering to Restangular configurations.

See Restangular error handler documentation.

Methods

  • init()

    Initializer method, to be called when module is loaded.

    It registers error interceptors to all configured Restangular configuration services. See restangularConfig property.
